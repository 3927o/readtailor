# 裁读 ReadTailor · 设计风格指南

> 落地页（landing page）的视觉与文案风格规范。可作为给 AI / 设计师的参考，
> 让新页面、新物料与现有产品气质保持一致。

---

## 一、整体气质（定调）

**「一封安静的信」**——克制、文学、editorial（杂志 / 书籍编辑感）的高级感。

不是科技 SaaS 的炫酷，也不是中式古风，而是**当代精装书 + 独立杂志内页**的感觉：
大量留白、衬线大字、墨绿点缀、逐字浮现的动效。整站像一个人在低声、诚恳地写信给读者，
而不是产品在「推销功能」。

关键词：**editorial · 文学感 · 克制 · 温润 · 留白 · 手作感 · premium 而非 retro**。

---

## 二、色彩系统

「暖白纸 + 墨黑字 + 鼠尾草绿」的三色极简体系。**唯一的彩色是墨绿**，其余全是中性灰阶。

| 角色 | Token | 值 | 用途 |
|---|---|---|---|
| 背景主色 | `--lt-bg` | `#FAFAF6` | 暖白／米白纸张感，**不是纯白** |
| 背景次色 | `--lt-bg-2` | `#F0EFE9` | 浅灰卡片底（如「补全」块） |
| 卡片白 | `--lt-bg-card` | `#FFFFFF` | 手机屏内 |
| 主文字 | `--lt-ink` | `#0A0A09` | 近黑、带一点暖，**不是纯黑** |
| 次文字 | `--lt-ink-2` | `#4E4D49` | 正文 / 说明 |
| 弱文字 | `--lt-ink-3` | `#8E8C85` | 标签 / 提示 / 占位符 |
| **品牌绿** | `--lt-green` | `#2F6A52` | 鼠尾草 / 松针绿，强调色 |
| 深绿 | `--lt-green-deep` | `#1F4D3A` | 斜体强调字、按钮 hover |
| 浅绿底 | `--lt-green-soft` | `rgba(47,106,82,0.10)` | 注释卡背景、选中态 |
| 分隔线 | `--lt-rule` | `rgba(10,10,9,0.12)` | 细线、规则线 |
| 分隔线弱 | `--lt-rule-2` | `rgba(10,10,9,0.06)` | 更轻的下划线 / 虚线 |
| 错误红 | — | `#b4452f` | 仅用于表单报错，砖红而非鲜红 |

**用色原则**：90% 画面是暖白与黑灰，绿色只用在「需要被点亮」的地方——
强调词、引导卡左边框、选中按钮、CTA 按钮、进度点。**绿色永远是「点睛」而非「铺面」**。

## 三、字体系统（核心特征：双声部 / Two-Voice Type）

最有辨识度的设计决策——**两种声音用两种字**：

1. **「信」的大字 = 衬线明朝体**
   `--lt-serif: 'LanternMingA'(上元明朝), 'Noto Serif SC', 'Songti SC', Georgia`
   用于所有叙述句、标题、正文。气质：文学、克制、有呼吸感的现代明朝体（**不是古风楷体**）。

2. **「演示 UI / 手机壳内」= 未来荧黑（黑体）**
   `--lt-demo: 'Glow Sans SC', 'Noto Sans SC', 'PingFang SC'`
   凡是进入手机界面、可交互控件、产品 UI 的部分，字体局部切换成黑体，
   制造「这是产品、那是信」的声部区分。实现上是在 `.lt-phone` 等作用域内
   把 `--lt-serif/--lt-sans/--lt-mono` 全部重指向黑体，子规则无需 `!important`。

3. **「编辑铭牌」标签 = 等宽体**
   `--lt-mono: 'JetBrains Mono'`
   用于 kicker 小标签、导航期号、提示文字。**全大写 + 大字距**
   （letter-spacing 0.12–0.22em），营造杂志栏目 / 期刊编号的 editorial 感。

**排版细节**：
- 大字 `clamp(28px, 3.4vw, 42px)`，行高 1.5–1.55，字距 `-0.005em`（微收紧）。
- 强调词用 `<em>` 斜体 + 深绿色。
- 正文阅读区行高高达 1.9–1.95，段落首行缩进 2em（中文书籍排版习惯）。
- `text-wrap: pretty` 避免孤行。

## 四、布局与空间

- **全屏单句叙事**：整站是「一屏一句」的离散翻页（discrete paging），不是连续滚动。
  每次滚动 / 方向键 / 空格 = 推进一句。body 锁滚动，舞台占满视口、`place-items: center` 居中。
- **极大留白**：舞台 padding `110px 64px`，内容最大宽度 `38ch`（叙述句）/
  `1000px`（演示）/ `720px`（信）。文字永远不顶满，四周大量空气。
- **顶部细节铭牌**（masthead）：左 `裁读 · ReadTailor`，右 `一封信 · 致一直读不完那本书的你`，
  带毛玻璃 `backdrop-filter: blur(10px)` + 底部细线。
- **底部消隐**：`::after` 一道 120px 渐变把离场文字溶进背景，强化「信纸」的轻盈。
- **进度系统**：顶部 2px 墨绿进度条 + 底部居中的迷你圆点导航
  （当前点拉长成胶囊 + 绿色光晕，特殊场景点旋转 45° 成菱形）。

## 五、动效（克制但讲究）

- **逐字浮出**（招牌动效）：叙述句每个字 `opacity 0→1` + `translateY(0.24em→0)` +
  `blur(2px→0)`，每字错开约 55ms，标点处停顿更久。缓动 `cubic-bezier(.2,.7,.2,1)`，
  时长约 1100ms。像信在被一笔一画写出来。
- **两拍揭示**（two-beat）：演示标题先作为普通句子居中出现（beat 0），推进后标题滑到左上、
  说明淡入、手机从右侧滑入（beat 1，FLIP 动画）。
- **交互场景**用更平静的整块淡入淡出（无位移、无模糊），区别于「句子」的逐字动效——
  **控件不该像句子那样表演**。
- 全站尊重 `prefers-reduced-motion`，关闭所有动画。

动效哲学：**慢、柔、有起落**，缓动几乎都用同一条 ease-out 曲线（`cubic-bezier(.2,.7,.2,1)`）保持一致性。

## 六、组件与图形语言

- **注释卡**（产品核心）：浅绿底 + **左侧 2px 实心绿边框** + 右上圆角 `0 4px 4px 0`，
  是反复出现的标志性容器（导读卡、铺路注释、CTA 致谢）。
- **kicker 小标签**：等宽全大写 + 前置一道 28px 绿色短横线（`::before`），杂志栏目头风格。
- **手机壳**：深色 `#1A1916` 机身、38px 圆角、刘海 + Home 条、柔和多层投影，
  内部是真实的阅读 App 界面演示。
- **气泡 / 长按**：墨黑底白字气泡 + 小三角，模拟 AI 陪读的唤起。
- **CTA = 一封回信**：用中文书信的「题词 → 称呼 → 正文 → 署名」节奏，填空用虚线下划线输入框，
  按钮是绿色全圆角胶囊。**没有传统表单的盒子感**，全靠排版节奏。
- 装饰元素：细规则线（rule）、`·` 间隔点、`——` 破折号、`⌜ ⌟` 夹注角标——
  都是**印刷 / 书信**而非 UI 图标的语汇。几乎不用 emoji 和彩色插画。

---

## 七、文案 / 语气

- **第二人称、低声、像私信**：「你好，我的朋友。」「你愿意，我们就开始。」
- 大量**比喻而非功能罗列**：把难读的书比作「没有路标的森林」，产品是「向导」而非「替你走的人」。
- 句子短、留白多、用换行控制呼吸。
- 落款式收尾：`—— ReadTailor · 裁读 / 2026`。
- 中英混排，英文作为副标题 / 期号的 editorial 装饰（`· ReadTailor`、`ISSUE` 感）。

---

## 八、给 AI 的速记 prompt（可直接粘贴）

> 设计风格：当代 editorial / 文学杂志内页气质，安静克制的「一封信」。暖白纸底 `#FAFAF6` +
> 近黑暖墨 `#0A0A09` + 唯一强调色鼠尾草墨绿 `#2F6A52`。双字体声部：叙述与标题用**现代明朝衬线体**
> （非古风），产品 UI / 控件用**几何黑体**，小标签用**等宽体全大写大字距**。极大留白、一屏一句、
> 内容窄栏居中。标志性元素：浅绿底 + 左绿边框的注释卡、前置短横线的 kicker 小标签、逐字浮现的文字动效、
> 虚线下划线的填空输入。动效慢而柔（ease-out `cubic-bezier(.2,.7,.2,1)`）。语气第二人称、像私信、
> 多用比喻少用功能词。避免：纯白纯黑、古风楷体、多彩渐变、emoji、SaaS 式卡片阴影堆叠、营销话术。
